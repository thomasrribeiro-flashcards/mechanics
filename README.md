# mechanics

Flashcards for Classical Mechanics — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

Card creation and auditing follow the deck-specific decisions in
[CARD_README.md](CARD_README.md) and the canonical standards in the flashcards
application repository. The deck complements—not replaces—solving novel mixed
mechanics problems on paper.

## Scope

- Subject: `physics`
- Level: introductory, calculus-aware classical mechanics
- Prerequisites: algebra, trigonometry, and calculus concepts as introduced
- Mastery enables: model selection, representation translation, qualitative
  prediction, and representative mechanics problem solving
- Excludes: continuum mechanics, fluids, relativity, quantum mechanics, and
  advanced analytical mechanics

## Coverage

| File | Topic |
|------|-------|
| 01_foundations | Scientific method, SI units, dimensional analysis, sig figs |
| 02_vectors | Scalars/vectors, components, addition, dot product, cross product |
| 03_kinematics_1d | Position/velocity/acceleration, kinematic equations, free fall |
| 04_kinematics_2d | Projectile motion, circular motion, relative motion |
| 05_newtons_laws | Newton's 1st/2nd/3rd laws, inertia, inertial frames, FBDs |
| 06_forces | Weight, normal, tension, static/kinetic friction, spring, drag |
| 07_work_energy | Work, work-energy theorem, KE/PE, energy conservation, power |
| 08_momentum | Momentum, impulse, elastic/inelastic collisions, center of mass |
| 09_rotation | Angular kinematics, torque, moment of inertia, angular momentum |
| 10_gravitation | Newton's law of gravity, orbits, escape velocity, Kepler's laws |
| 11_oscillations | SHM, spring-mass, pendulum, energy, damping, resonance |
| 12_synthesis | Model selection, representation translation, mixed-method problems |
